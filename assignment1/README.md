# Assignment 1: Pthreads Programming

This project explores parallel-programming concepts using POSIX Threads. It includes exercises on polynomial multiplication, shared variable updates, false sharing, bank simulations, and synchronization barriers.

## 1. Prerequisites

Python 3 is required for the plotting scripts. Install the helper libraries with:

```bash
pip install pandas matplotlib
```

## 2. How to Build

From the `assignment1/` directory run:

```bash
make
```

Executables are written to `bin/`. Clean the workspace with:

```bash
make clean
```

## 3. How to Run Experiments & Generate Plots

Run the shell script for each exercise to generate the CSV data first, then run the matching Python script to create the plots. Make sure the scripts are executable (once per clone) with:

```bash
chmod +x scripts/*.sh
```

All commands below assume you are inside `assignment1/`.

### Exercise 1 – Polynomial Multiplication

Run experiments:

```bash
./scripts/run_experiments.sh
```

Output: `results/raw_results.csv`

Generate plots:

```bash
python3 src/plot_results.py results/raw_results.csv
```

### Exercise 2 – Shared Variable Update

## 4. Directory Structure

- `src/` – C sources and plotting scripts
- `scripts/` – Shell helpers for running experiments
- `bin/` – Compiled executables (created by `make`)
- `results/` – CSV data and PNG plots generated by the scripts
- `Makefile` – Build rules for all binaries

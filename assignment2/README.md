# Assignment 2: OpenMP Programming

This project explores parallel-programming concepts using OpenMP Programming. It includes exercises on multiplying polynomials, sparse matrix-vector multiplication and parallel mergesort.

## 1. Prerequisites

Python 3 is required for the plotting scripts. Install the helper libraries with:

```bash
pip install pandas matplotlib
```

## 2. How to Build

From the `assignment2/` directory run:

```bash
make
```

Executables are written to `bin/`. Clean the workspace with:

```bash
make clean
```

## 3. How to Run Experiments & Generate Plots

Run the shell script for each exercise to generate the CSV data first, then run the matching Python script to create the plots. Make sure the scripts are executable (once per clone) with:

```bash
chmod +x scripts/*.sh
```

All commands below assume you are inside `assignment2/`.

### Exercise 1 – Polynomial Multiplication

Run experiments:

```bash
./scripts/re_poly_mult.sh
```

Output: `results/results.csv`

Generate plots:

```bash
python3 src/plot_results.py results/results.csv
```

### Exercise 2 – Sparse Matrix-Vector Multiplication

Run experiments:

```bash
./scripts/sparse_mat_script.sh
```

Output: `results/raw_results.csv`

Generate plots:

```bash
python3 src/sparse_mat_plot.py results/raw_results.csv
```

### Exercise 3 – Parallel Mergesort

Run experiments:

```bash
./scripts/re_mergesort.sh
```

Output: `results/results3.csv`

Generate plots:

```bash
python3 src/plot_mergesort.py results/results3.csv
```

## 4. Directory Structure

- `src/` – C sources and plotting scripts
- `scripts/` – Shell helpers for running experiments
- `bin/` – Compiled executables (created by `make`)
- `results/` – CSV data and PNG plots generated by the scripts
- `Makefile` – Build rules for all binaries
